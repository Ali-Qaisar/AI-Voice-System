# Friday AI Assistant

Voice-activated AI assistant powered by online AI services.

## 🚀 Quick Start

```bash
# 1. Install
pip install -r requirements.txt

# 2. Configure (already done!)
# Your API key is configured in .env

# 3. Run
python main.py
```

Say "Friday" to activate, then speak your command!
